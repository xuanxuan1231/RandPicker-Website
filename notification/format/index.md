# 通知格式

RandPicker 的通知格式可以被自定义，不同通知方案的格式是独立的。比方说，“系统原生”的通知格式可以和“ClassIsland”的通知格式不同。

## 设置通知格式

::: v-pre
RandPicker 的通知格式中包含 `{{foo}}` 的内容 (如 `{{names}}`，下文称作<mark>“标识符”</mark>)。发送通知时，RandPicker 会把它们替换成实际的数据 (如 `{{names}}` 替换为 学生姓名/小组名称的列表)。
:::

RandPicker 提供以下标识符：

|标识符|数据|
|-|-|
|names|姓名/小组名列表|
|...|...|

## 数据选项

RandPicker 针对特定数据提供了显示选项。如用户可以设置“姓名/小组名列表”中，每个名字间的分隔符 (张三<mark>,</mark>李四)。
