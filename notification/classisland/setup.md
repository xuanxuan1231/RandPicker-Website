# 安装

首先，确保你的电脑符合 [先决条件](index.md#requirements)。

然后，遵循 [ClassIsland 文档](https://docs.classisland.tech/app/plugin.html)，在【插件市场】中<mark>安装“RandPicker”插件</mark>。

:::info 别急
因为插件编写尚未完全完成，且插件市场审核需要时间，现在还不能在【插件市场】中找到“RandPicker”插件。
:::

一旦 RandPicker 与 ClassIsland 建立连接，你就可以在 RandPicker 设置 -> 通知 & 集成 -> ClassIsland 中看到这条消息：

![success](success.png)

至此，你已完成 ClassIsland 集成的安装。
